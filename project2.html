Project Overview¶

For this project, I'll analyze a large bipartite dataset of book readers. Nodes are either readers (top nodes) or books (bottom nodes). This is a large dataset (271,000 books and 278,000 readers). To save processing time, only the first 20,000 of each will be analyzed. Readers have been deidentified but have location and ages provided. Books are listed by ISBN. The main goal is to identify similar books based on the readers who read them.

Soure: http://www.informatik.uni-freiburg.de/~cziegler/BX/

def fetch_data(fn):
    if not os.path.exists(fn):
        print('Downloading', fn)
        try:
            r = requests.get('https://github.com/mmippolito/cuny_data620_project2/blob/main/' + fn + '?raw=true').text
            fh = open(fn, 'w', encoding='utf-8')
            fh.write(r)
            fh.close()
        except:
            return 1
    return 0

# Define function to trim edges based on weight; modified from source:
# Maksim Tsvetovat & Alexander Kouznetsov, Social Network Analysis for Startups, 2011
def trim_edges(g, weight=1):
    g2 = nx.Graph()
    for f, to, edata in g.edges(data=True):
        if edata['weight'] > weight:
            if f not in g2.nodes:
                if g.nodes(data=True)[f]['bipartite'] == 0:
                    g2.add_node(f, loc=g.nodes(data=True)[f]['loc'], age=g.nodes(data=True)[f]['age'], bipartite=0)
                else:
                    g2.add_node(f, title=g.nodes(data=True)[f]['title'], author=g.nodes(data=True)[f]['author'], year=g.nodes(data=True)[f]['year'], pub=g.nodes(data=True)[f]['pub'], bipartite=1)
            if to not in g2.nodes:
                if g.nodes(data=True)[to]['bipartite'] == 0:
                    g2.add_node(to, loc=g.nodes(data=True)[to]['loc'], age=g.nodes(data=True)[to]['age'], bipartite=0)
                else:
                    g2.add_node(to, title=g.nodes(data=True)[to]['title'], author=g.nodes(data=True)[to]['author'], year=g.nodes(data=True)[to]['year'], pub=g.nodes(data=True)[to]['pub'], bipartite=1)
            g2.add_edge(f, to, weight=edata['weight'])
    return g2

# Function to display books in a data frame
def bookToDf(centtmp, sgtmp, metricName):
    # cent=sorted list of centrality metrics
    # sg=subgraph
    # metricName=the name of the metric, e.g. degree, closeness, etc

    # first create empty data frame
    df = pd.DataFrame(columns=['Rank', metricName, 'isbn', 'title', 'author', 'year', 'pub', 'bipartite'])
    
    # loop through each centrality entry, adding each one to the data frame
    for i, b in enumerate(centtmp):
        if i > 9: break    # only use the top ten
        if metricName == 'degree':    # degree uses tuples instead of scalars
            tmpd = {'rank': [i+1], metricName: [b[1]], 'isbn': [b[0]]}    # add rank, the metric, and isbn
            tmpd2 = sgtmp.nodes(data=True)[b[0]].copy()                   # append the 'data' values to the dict
        else:
            tmpd = {'rank': [i+1], metricName: [cent[b]], 'isbn': [b]}    # add rank, the metric, and isbn
            tmpd2 = sgtmp.nodes(data=True)[b].copy()                      # append the 'data' values to the dict
        # Dataframe.from_dict needs an array of values, so loop through keys in the dict and make each one a one-elemnt array
        for k in tmpd2.keys():
            tmpval = tmpd2[k]    # temporarily store the value since we'll be overwriting it with an empty list
            tmpd2[k] = []        # create an empty list
            tmpd2[k].append(tmpval)    # now put the value back as the first (and only) element of the list
        tmpd.update(tmpd2)
        df = pd.concat([df, pd.DataFrame.from_dict(tmpd)], ignore_index=True)
    print()
    display(df)

Analysis and conclusions¶
Based on the results, a few conclusions can be drawn:


    

  1. Partitioning the network into subgraphs makes it clear that the groups were separated by the readers' language (English for the large subgraph, French for the small one).
    2. 

  2. In the large subgraph, Snow Falling on Cedars and Divine Secrets of the Ya-Ya Sisterhood were in the top four for all centrality degree metrics, which suggests that they were widely read.
    3. 

  3. Divine Secrets of the Ya-Ya Sisterhood had the highest eigenvector centrality, indicating that it was read by the most prolific readers.
    4. 

  4. Wild Animus had the highest betweenness centrality, suggesting that it was a cross-over book of sorts that possibly spanned genres.
    5. 

  5. Snow Falling on Cedars tops the closeness centrality chart, so it can be infered that it was the one that appealed to a wide audience.
    6. 

  6. In the small subgraph, Orwell's translation of Animal Farm (La Ferme Des Animaux) ranked either first or second by all centrality metrics.
    7. 

  7. Clique sizes within the large subgroup are normally distributed with a mean of 38.9, which could mean that most readers on the site review about that many books, or that the same several readers review about that many books.
    8. 

  8. The ego graphs of the top books confirm that the readers are, indeed, separated by language, as infered by geographic location of the reader.
    9. 

  9. However, there does not seem to be the same kind of correlation among readers based on their age or their region of the US.
    10.
